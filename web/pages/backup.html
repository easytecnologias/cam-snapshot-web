Base conectada ao servidor e ao inventario do SightOps.

  • UrBackup Server online no host

Proximas etapas
Ordem recomendada para colocar em operacao.
  1. Entrar no EASY Backup.
  2. Baixar o Cliente Windows pelo painel.
  3. Executar em uma maquina piloto como Administrador.
  4. Sincronizar UrBackup e importar Windows.
  5. Fazer um backup e um restore de teste.
